# DevTrack Frontend TO visit devtrack just go to devtrack.dev

Frontend for **DevTrackJR**, a shift and production tracking app built with **React** and **Vite**. Connects to the backend API for authentication and production management.

---

## Features

- User login/logout with JWT stored in HTTP-only cookies  
- Track shifts, production, pallet returns, and downtime  
- Responsive and clean UI  
- Communicates securely with backend API  

---

## Tech Stack

### Backend(Backend is in Another Repository)

- Python
- Django
- Django REST Framework
- JWT Authentication
- SQLite

### Frontend

- React
- React Router
- Chakra UI
- Fetch API
- Toast notifications
